# Vietnam FINPLAN Project
This folder contains FINPLAN model files for aggregated renewable power plants under the Power Development Plan 8 (PDP8), including solar (grounded, floating), wind (onshore, near shore, offshore), biomass, and waste-to-energy.
All FINPLAN model files are balanced. The input data and results for each file are also included.

## Contents
- **vn_solgro:** solar (grounded)
- **vn_solflo:** solar (floating)
- **vn_winons:** wind (onshore)
- **vn_winnea:** wind (near shore)
- **vn_winoff:** wind (offshore)
- **vn_bio:** biomass
- **vn_wte:** waste-to-energy

More details on each scenario can be found via the citation.
